# My Portfolio Template

> 나만의 깃허브 포트폴리오 템플릿

### 🚀 Getting Started

1. **Node.js & npm 확인**  
   공식 [Node.js LTS](https://nodejs.org/)를 설치 후 버전 확인:
   ```bash
   node -v  # >=16.x
   npm -v   # >=8.x
   ```

2. **프로젝트 생성 (Vite + React)**  
   Vite로 초기화하고 의존성 설치:
   ```bash
   cd my-portfolio
   npm install
   ```

3. **개발 서버 실행**
   ```bash
   npm run dev
   ```
